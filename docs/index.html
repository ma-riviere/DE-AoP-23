Description

This website showcases the data and analyses of our paper:

“Apnea of Prematurity induces short and long-term development-related transcriptional changes in the murine cerebellum”

The goal is to allow interested parties to explore our data and see the output of our analyses without running the code themselves.

If you wish to access the code of the analyses themselves, head to the main branch of our repository.

You can find more information on the overarching project that lead to this paper on the first author’s website.

Structure of this website

This website is divided into 4 major sections, accessible through the top navigation menu:

  1. Data: Explore and download the paper’s data.

  2. Analysis: Our paper’s code and its output, organized as documentation.

  3. Paper: The paper’s PDF, abstract, and citation data.

  4. Extras: Talks, posters and figures related to our paper and its overarching project.
